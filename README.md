<summary><strong>Stress Detection System</strong></summary>

A real-time emotion + stress level detection system built using **Keras (CNN Model)**, **OpenCV**, and a lightweight **Eel web UI**.  
This project captures frames from a webcam, detects faces, predicts emotional states using a trained CNN model, and maps those emotions to stress levels (**high**, **low**, **none**).
